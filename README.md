# ARTEcatering

Sito web di ARTEcatering – catering professionale per eventi aziendali e privati.

## Struttura

- **index.html** – Pagina principale (layout a due colonne)
- **assets/css/** – File CSS modulari (variabili, reset, tipografia, layout, componenti, responsive)
- **assets/js/app.js** – JavaScript vanilla (galleria lightbox + smooth scroll)
- **assets/css/fontawesome-all.min.css** – Icone Font Awesome
- **images/** – Immagini del sito e galleria
